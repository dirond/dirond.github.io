    SMB

    Создан Суббота 19 ноября 2022

    SMB - Server message block protocol. Используется для предоставления/получения доступа к различным ресурсам, каталоги, файлы в них, принтеры и т.п. Работает на 445 (ранние версии на 139) порту TCP.

    Инструменты:

    enum4linux - Утилита для перебора сведений о сервисе работающим по протоколу SMB Smbclient - CLI - Утилита для работы с SMB smbget - Аналог wget для SMB ресурсов

    nse скрипты nmap для smb enumeration: nmap --script=smb-enum-shares.nse,smb-enum-users.nse -p $PORT $IP

    Metasploit модули: exploit/windows/smb/psexec
