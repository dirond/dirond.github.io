    venv

    Виртуальное окружение - сущность позволяющая изолировать проект в системе. Необходимо для того, чтобы не возникало конфликтов в зависимостях между различными проектами.

    Создание: 

    # Создание окружения
python3 -m venv /path/to/project
# Активация окружения
source /path/to/project/bin/activate
